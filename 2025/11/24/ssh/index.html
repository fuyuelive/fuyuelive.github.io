ssh步骤

先装好ubuntu或安装基于linux内核而衍生出来的系统

确保linux已经安装并成功启动ssh服务（如以下命令错误就把sshd改成ssh）

在命令终端输入service sshd status（sudo service sshd status）或systemctl status sshd（sudo systemctl status ssh）查看是否启动了ssh服务，当出现active（running）时，则启动。

更新软件仓库（apt-get update），完成后，使用apt-get install openssh-server安装openssh服务，若ssh服务已安装，但未启动，使用systemectl start sshd（service sshd start）启动，启动后进行后续操作。

查看ip地址

ifconfig，ufw status（查看防火墙状态），ufw disabled（禁用防火墙），ufw enabled（启动防火墙）。

在本地使用ssh命令连接服务器，在本地命令行输入ssh [用户名]@[ip地址]

传输文件

scp [选项] [源文件/目录] [用户名]@[目标地址]:[目标路径]

示例：scp -r C:\Users\HP\Desktop\sample.txt ubuntu@192.168.230.128:/home/ubuntu
