Linux常用命令4：系统管理篇

在Linux系统中，了解系统状态和管理进程是系统管理员的重要职责。本文将介绍查看系统信息和管理进程的常用命令，帮助你监控和维护Linux系统。

六、使用系统信息类命令

1、dmesg（显示连接到系统上的设备）

用实例名和物理名称来标识连到系统上的设备，也用于显示系统诊断信息、操作系统版本号、物理内存大小以及其他信息

*2、free（查看内存占用情况）

查看系统内存、虚拟内存的大小及占用情况

3、date（显示时间）

查看系统当前日期和时间，也可加参数-d来设置当前日期和时间（月日年）

4、cal（日历）

显示指定月份或年份的日历，可以带两个参数，其中，年、 月份用数字表示；只有一个参数时表示年份，年份的范围为1~9999；不带任何参数的cal命令显示当前月份的日历。

5、vmstat（查看硬件统计信息）

提供关于系统虚拟内存、进程、CPU、I/O的统计信息，可以用来观察系统的整体性能。

*6、iostat（磁盘监视）

用于IO的监控，显示磁盘的读写速度。（需安装apt install sysstat）

*7、df（查看磁盘使用情况）

*8、netstat（查看网络信息）

七、使用进程管理类命令

*1、命令 &（将命令放在后台执行）

回显一个作业号和进程号。

使用jobs命令可以查看终端启动了哪些作业-l时长格式显示结果

使用fg +作业号可将后台转为前台运行

ctrl+Z暂停前台运行的作业

使用命令bg +作业号可将暂停的前台运行的作业切换到后台

*2、ps（查看进程）

参数：-lf：详细信息

*3、kill（终止程序，常配合ps使用）

参数：-l： 查看可发送的信号有哪些

4、pidof [参数] [服务名称]（查看进程pid值）

5、killall [参数] [进程名称]（终止指定名称下的所有进程）

6、top（监控进程状况）

监控进程状况（默认5秒刷新）也可用“top -d 20”改为每二十秒刷新一次

shift+m对内存情况排序

总结

本文介绍了Linux系统中查看系统信息和管理进程的常用命令。通过这些命令，你可以了解系统的硬件状态、内存使用情况、磁盘使用情况，以及管理系统中的进程。掌握这些命令对于系统监控和维护非常重要。在后续文章中，我们将继续介绍其他常用命令，敬请期待！
